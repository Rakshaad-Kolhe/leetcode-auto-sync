# Product Roadmap 🗺️

Future goals for **LeetCode Auto Sync**:

## Version 1.0.0 (Current)
- [x] Centralized JSON configuration system.
- [x] Custom README markdown templates (`classic`, `detailed`, `minimal`).
- [x] Configurable folder layout strategies.
- [x] Intelligent synchronization engine with SHA-256 change detection.
- [x] Incremental topic page updates.
- [x] Health dashboard (`GET /status`) & diagnostic support bundles (`GET /diagnostics`).
- [x] Automated release packaging (`extension.zip`, `pyproject.toml`).

## Version 1.1.0 (Planned)
- [ ] Multi-repository synchronization support.
- [ ] Automated Web Store extension updates.
- [ ] Custom badge designer for problem READMEs.

## Version 2.0.0 (Future)
- [ ] Cloud sync backup integration.
- [ ] Team & group submission dashboards.
